## The Problem

The CARES Act created urgent need for state governments to identify and reach eligible businesses quickly. The challenge: the data existed across multiple systems, but there was no infrastructure to aggregate, clean, and analyze it at the scale required.

Manual processes that might have taken weeks would leave eligible businesses without relief. Speed was critical.

## The Solution

Built an end-to-end data pipeline using Alteryx and Power BI that could process large volumes of messy government data into actionable intelligence.

Key components:
- **Alteryx workflows** automating data ingestion, cleaning, and transformation from multiple source systems
- **Power BI dashboards** giving program administrators real-time visibility into eligibility and outreach status
- **Data pipeline architecture** designed for repeatability — the same process could be rerun as new data arrived
- **Quality assurance process** to validate account identification accuracy before outreach

## The Results

- Identified 20,000 eligible accounts representing $400M in economic opportunity
- Saved an estimated 10,000 hours of manual data processing work
- Delivered results fast enough to meaningfully impact CARES Act relief distribution
- Created a reusable data infrastructure for future program administration
